# התאמות ותמיכה במובייל עבור מערכת מלון דיאם

## מה נעשה?

בוצעו מספר שיפורים והתאמות למערכת על מנת שתתמוך בצורה טובה יותר במכשירים ניידים וכן לאפשר פונקציונליות PWA (Progressive Web App).

## שינויים עיקריים

### 1. תמיכה ב-PWA (Progressive Web App)

* עודכן קובץ `manifest.json` כולל הגדרות נכונות עבור אייקונים, צבעים וגדלים
* נוסף קובץ `service-worker.js` לתמיכה בשימוש באפליקציה במצב offline
* עודכן קובץ `index.html` עם meta tags מתאימים לתמיכה ב-PWA ובהתקנה כאפליקציה:
  * `apple-mobile-web-app-capable`
  * `apple-mobile-web-app-status-bar-style`
  * `viewport` משופר

### 2. ממשק משתמש רספונסיבי

* סרגל צד מותאם למובייל:
  * בוצעה החלפה של הסרגל הצדדי הקבוע בתפריט צף בגרסת מובייל
  * נוסף ניווט תחתון (bottom navigation) עם הפעולות החשובות ביותר
  * נוסף כפתור המבורגר לפתיחת התפריט המלא

* לייאאוט כללי והתאמות:
  * עודכנו ריווחים והגדרות רוחב להתאמה אוטומטית
  * נוספו שוליים תחתונים להתחשבות בניווט התחתון
  * הוספת חוקי CSS ספציפיים למסכים קטנים

### 3. דף דאשבורד

* הותאם למסכי מובייל:
  * שיפור תצוגת הכרטיסים של המודולים
  * התאמת גדלי טקסט וריווחים
  * שיפור הניווט בין התאריכים למובייל

### 4. דף הזמנות

* שיפורים בדף ניהול ההזמנות:
  * סרגל חיפוש קומפקטי למובייל
  * הוספת `FAB` (Floating Action Button) לתוספת הזמנה חדשה
  * התאמות לוח השנה עם מימדים מותאמים למסכים קטנים
  * התאמת גודל תאים ופונטים
  * גלילה אופקית משופרת עם טיפול בהצגת תוכן הזמנה גם בשטח מצומצם

### 5. טופס הזמנות

* רה-ארגון מבני של הטופס:
  * הוספת לשוניות (tabs) במובייל לחלוקת הטופס לחלקים קטנים יותר
  * התאמת שדות הזנה לתצוגה של מסך קטן
  * התאמת כפתורי פעולה למובייל בתחתית הטופס
  * יכולת הצגת מידע חלקי בהתאם לשטח המסך (טקסט מקוצר, אייקונים בלבד וכו')

## טכנולוגיות וכלים

* שימוש ב-`useMediaQuery` של Material UI לזיהוי גודל מסך
* הגדרות CSS דינמיות לשליטה בתצוגה
* Service Worker לתמיכה ב-PWA
* שינויים בקומפוננטות מבוססות grid לתצוגה מיטבית

## איך להשתמש?

המערכת כעת נטענת באופן אוטומטי בגרסה המתאימה למכשיר:
* במחשב רגיל: גרסה רחבה עם תפריט צד
* במכשיר נייד: גרסה מותאמת עם ניווט תחתון ותפריט נשלף

בנוסף, ניתן כעת להתקין את המערכת כאפליקציה על מכשירים ניידים (יופיע באופן אוטומטי הצעה בדפדפן בהתאם למכשיר).
